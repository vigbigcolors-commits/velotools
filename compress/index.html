📊 Quality guide

90–100%

PNG is always lossless — quality slider has no effect on PNG output. To reduce PNG file size, use Convert → WebP (25–35% smaller, full alpha transparency preserved).

🔄 Format guide

WebP Best for websites
Google's format. 25–40% smaller than JPG/PNG at equal quality. Full transparency. All modern browsers since 2020.
AVIF Smallest files
Newest standard. Up to 50% smaller than JPEG. HDR support. Chrome, Firefox, Safari 2022+.
JPEG / JPG
Universal format for photographs. Lossy — detail loss invisible at 80%+. No transparency. Works everywhere.
PNG
Lossless, zero quality loss. Full alpha transparency. Ideal for logos, screenshots, sharp-edge graphics.
