How PDF merging works (and why order matters)

When you merge PDFs, you are appending page trees from each source document into a new file. VeloTools uses lossless page copy — each page is transferred with its original vector text, embedded fonts, and image streams intact. Nothing is re-rendered to JPEG, so a 50-page report followed by a 3-page contract produces a 53-page PDF that looks identical to printing each file and stapling them in order.

File order is the merge order. If you drop cover.pdf, appendix.pdf, and terms.pdf, the output runs cover → appendix → terms. Use drag-and-drop on the file list to fix order before merging — there is no undo after download.

Merge vs compress — different jobs

Merging combines documents; compressing shrinks them. A common mistake is running merge on already-compressed scans, then compressing the result — that works, but merge itself does not reduce file size. If your goal is one smaller email attachment, merge first, then use our PDF Compressor on the combined file.

Password-protected PDFs

Encrypted PDFs cannot be merged until they are unlocked. If merge fails with a password error, open the file in Unlock PDF (you must know the password), download the unlocked copy, then merge.

Privacy architecture

Server-based merge tools upload every file to cloud storage. VeloTools reads PDFs with the browser File API and combines them with pdf-lib in local memory. No network request is made. Suitable for contracts, medical records, and financial statements you would never send to a third party.

Frequently asked questions

Is merging PDFs online safe for confidential documents?
Yes — with VeloTools, your files never leave your device. There is no server processing step to breach. After the page loads, you can disconnect from the internet and merge still works.
Does merge change image or text quality?
No. Pages are copied as-is. Vector text remains vector; photos keep their original compression. This is different from tools that rasterize every page to images.
What is the maximum number of files?
Up to 30 PDFs per merge. Total size is limited by your device RAM — most users merge dozens of pages without issue.
Can I merge PDFs on iPhone or Android?
Yes. Modern mobile browsers support the File API and pdf-lib. Use the file picker to select multiple PDFs from Files or Google Drive.
