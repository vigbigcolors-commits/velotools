Split vs extract: what this tool does

Split creates new PDF documents from selected pages using lossless page copy — fonts and images stay intact. Per-page mode outputs page_1.pdf, page_2.pdf, etc. Range mode pulls contiguous or scattered pages into one new file (e.g. pages 1–3 and 5 → one PDF).

When to split before compressing

Large scanned PDFs sometimes fail to compress in one pass on low-memory devices. Split into chunks, compress each with Compress PDF, then merge again if needed.

Privacy

Your document is read with the File API and processed with pdf-lib locally. No bytes are transmitted over the network.
